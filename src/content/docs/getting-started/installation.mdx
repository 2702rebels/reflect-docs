Reflect is available
- online at [reflect.2702rebels.com](https://reflect.2702rebels.com/)
- as a desktop application supporting Windows, MacOS and Linux platforms.

[Download the latest release](https://github.com/2702rebels/reflect/releases/latest) or use one of the following shortcuts:

| Platform                  | URL  |
| ------------------------- | ---- |
| Windows x64 (Standalone)  | <a href={`https://github.com/2702rebels/reflect/releases/download/v${RELEASE_VERSION}/Reflect-${RELEASE_VERSION}.exe`}>Reflect-{RELEASE_VERSION}.exe</a> |
| Windows x64 (Installer)   | <a href={`https://github.com/2702rebels/reflect/releases/download/v${RELEASE_VERSION}/Reflect-${RELEASE_VERSION}-win-x64-setup.exe`}>Reflect-{RELEASE_VERSION}-win-x64-setup.exe</a> |
| Windows ARM64 (Installer) | <a href={`https://github.com/2702rebels/reflect/releases/download/v${RELEASE_VERSION}/Reflect-${RELEASE_VERSION}-win-arm64-setup.exe`}>Reflect-{RELEASE_VERSION}-win-arm64-setup.exe</a> |
